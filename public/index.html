Champs vCard standards - Documentation

Cette application supporte tous les champs vCard (vCard 3.0) standards pour l'import/export de fichiers VCF.

Champs d'identité :

Champ vCard Description Exemple
NOM;FN Nom complet (valeur FN) Dupont
PRENOM Prénom Jean
TITLE Titre professionnel Dr.
PREFIX/PREFIX2 Préfixe d'honneur (HONORIFIC) Mme / M.
SUFFIX/SUFFIX2 Suffixe d'honneur Jr. / Sr.
NICKNAME Surnom Jeanne

Champs de dates :

Champ vCard Description Format attendu
BDAY Date d'anniversaire YYYY-MM-DD (ex: 1985-06-15)
BIRTHDAY-TIMEZONE Fuseau horaire de l'anniversaire (ex: Europe/Paris)

Champs d'organisation :

Champ vCard Description Exemple
ORGANIZATIONNAME/ORG Nom de l'organisation Ma Entreprise SAS
ROLE Rôle dans l'organisation CEO
NOTE Notes générales Téléphone portable : 06 12 34 56 78

Champs d'adresse :

Champ vCard Description Exemple
ADR Adresse (format structuré) Rue de la Paix 42, 75001 Paris
STREET/ADR-STREET Rue (composant adr) Rue de la Paix 42
LOCALITY/CITY Ville Paris
REGION/SUBDIVISION Région / département Île-de-France
POSTALCODE/ZIPCODE Code postal 75001
COUNTRY-REGION/COUNTRY Pays France

Champs familiaux :

Champ vCard Description Exemple
SPOUSE Conjoint(e) Sophie Dupont
FATHER/CHILD Père / Enfants Marc Dupont; Thomas; Julie

Tags personnalisés :

Vous pouvez ajouter des tags personnalisés via :

  • Le champ X-CUSTOM-TAGS dans les fichiers VCF
  • En séparant les tags par des point-virgules dans le formulaire (ex: "Client;VIP;Lien")

Note : Les formats de champs vCard peuvent varier selon l'application créatrice du fichier VCF. Cette application normalise les données à l'import.
